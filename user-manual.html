System Overview

QuickBill 305 is a comprehensive billing software designed to manage and automate billing processes for business operating permits and property rates. The system provides a complete solution for local assemblies to manage their revenue collection efficiently.

Key Features

Multi-User Role System

Five distinct user roles (Super Admin, Admin, Officer, Revenue Officer, Data Collector) each with specific permissions and access levels.

Intelligent Billing Automation

Smart bill calculation that automatically computes current bills based on business type, property structure, and room count using predefined fee structures.

GPS Integration

Precise location capture using Google Maps API with high accuracy mode, fallback options, and detailed address geocoding.

Payment Processing

Multiple payment methods including mobile money integration via PayStack, cash payments, and online transactions.

Public Portal

Online payment gateway for citizens to search bills, make payments, and download receipts instantly.

Comprehensive Reporting

Detailed analytics, audit trails, revenue reports, and defaulter management with export capabilities.

User Roles and Permissions

Super Admin

Full system access with restriction controls
The Super Admin has complete control over system restrictions and can override lockouts.
  • Set system restriction dates (1-3 months)
  • Configure warning alerts and countdown notifications
  • Manage all system settings and configurations
  • Override system restrictions and unlock operations
  • Access all features available to other roles

Admin

Full system access excluding restriction controls
Administrators manage day-to-day operations and system configuration.
  • Add and manage businesses and properties
  • Record payments and generate bills
  • Manage zones, sub-zones, and fee structures
  • Perform bulk operations and adjustments
  • Generate comprehensive reports and analytics
  • Send notifications and manage user accounts
  • Perform system backups and maintenance

Officer

Operational user with core business functions
Officers handle front-line operations and customer interactions.
  • Register new businesses and properties
  • Record payment transactions
  • View and edit business/property profiles
  • Generate and print individual bills
  • View map locations and GPS coordinates
  • Access limited reporting functions

Revenue Officer

Payment-focused role with location tracking
Revenue Officers specialize in payment collection and field operations.
  • Record payment transactions
  • Search and view account information
  • Access business and property maps
  • Generate daily collection summaries
  • View payment history and receipts

Data Collector

Data entry specialist with profile management
Data Collectors focus on accurate data capture and profile maintenance.
  • Register new businesses and properties
  • Edit existing profiles (limited permissions)
  • View business and property details
  • Access location maps for verification
  • Perform basic data entry functions

Business Management

Business Registration Process

The business registration form is organized into three main sections: Basic Information, Location Information, and Billing Information. Follow these steps carefully:

1
Navigate to Add Business

Go to Businesses → Add New from your dashboard menu. You'll see a comprehensive registration form with three distinct sections.

2
Basic Information Section

Required Fields:

  • Business Name * - Official name of the business
  • Owner Name * - Full name of the business owner
  • Business Type * - Use the type-ahead search feature (start typing to see suggestions)
  • Category * - Auto-populates after selecting business type

Optional Fields:

  • Telephone - Contact phone number (validated if provided)
  • Status - Defaults to "Active" (can be Active, Inactive, or Suspended)
3
Business Type and Category Selection (Type-Ahead Feature)

How the Type-Ahead Works:

  • Start typing the business type in the Business Type field
  • A dropdown appears showing matching business types from the database
  • Use arrow keys to navigate or mouse to select
  • Press Enter or click to select the business type
  • The Category dropdown automatically populates with categories specific to that business type
  • Select the appropriate category from the auto-populated options
  • The system immediately calculates and displays the current bill amount
Important: You must select a business type from the dropdown for the category to populate. Manual typing without selection will trigger a validation error.
4
Location Information Section

Location Capture Process:

  • Exact Location - Enter address manually or capture via GPS
  • Click "Capture GPS Location" button for automatic location capture
  • Allow location access when browser prompts
  • System attempts high-accuracy GPS first, then falls back to standard accuracy if needed
  • GPS coordinates (latitude and longitude) are automatically populated
  • Location address is reverse-geocoded to show neighborhood/area name
  • Coordinates can be manually edited if needed (validated: latitude -90 to 90, longitude -180 to 180)

Zone Assignment:

  • Zone * - Select from dropdown (required)
  • Sub-Zone - Optional, filtered by selected zone
GPS Tips: For best accuracy, ensure GPS is enabled, allow location access when prompted, and capture location while outdoors or near windows. The system provides detailed error messages if GPS capture fails.
5
Billing Information Section

Automatic Calculation Fields:

  • Current Bill - Automatically calculated based on selected business type and category (read-only field)
  • Amount Payable - Automatically calculated using formula: Old Bill + Arrears + Current Bill - Previous Payments (read-only field)

Manual Entry Fields (Optional):

  • Old Bill - Previous outstanding amount (defaults to 0.00)
  • Previous Payments - Amount already paid from previous bills (defaults to 0.00)
  • Arrears - Outstanding arrears from previous periods (defaults to 0.00)
  • Batch - Group identifier for batch processing
Dynamic Fee Display: As soon as you select a business type and category, a colored panel displays the current bill amount and description, helping you verify the correct fee structure is applied.
6
Form Validation and Submission

Before submission, the system validates:

  • All required fields have values
  • Business type was selected from the dropdown (not just typed)
  • Telephone format is valid (if provided)
  • GPS coordinates are within valid ranges (if provided)
  • Zone is selected
  • Business name doesn't already exist in the database

Click "Register Business" to save. Upon successful registration, you'll be redirected to the business profile view page.

Example: Restaurant Registration
1. Type "Rest" in Business Type field
2. Select "Restaurant" from dropdown
3. Category auto-populates with options like "First Class", "Second Class"
4. Select "First Class"
5. Current Bill automatically displays: GHS 500.00
6. Click GPS button, allow location access
7. Location captured: "East Legon, Accra" with coordinates 5.614818, -0.205874
8. Select Zone: "East Legon Zone"
9. Amount Payable auto-calculates: GHS 500.00 (assuming no old bills or arrears)
10. Submit form

Property Management

Property Registration Process

The property registration form is organized into four main sections: Owner Information, Property Details, Location Information, and Billing Information. Follow these steps:

1
Navigate to Add Property

Go to Properties → Add New from your dashboard menu.

2
Owner Information Section

Required Fields:

  • Owner Name * - Full name of the property owner

Optional Fields:

  • Telephone - Contact phone number (validated if provided)
  • Gender - Select from Male, Female, or Other
  • Ownership Type - Self (default), Family, Corporate, or Others
3
Property Details Section

Required Fields:

  • Structure * - Select from dropdown (e.g., Modern Building, Concrete Block, Mud Block)
  • Property Use * - Auto-populates based on structure (e.g., Residential, Commercial)
  • Number of Rooms * - Use +/- buttons or input directly (minimum 1 room)

Optional Fields:

  • Property Type - Modern (default) or Traditional
Room Counter Interface: The number of rooms field features an intuitive counter with minus (-) and plus (+) buttons. Click minus to decrease, plus to increase. Minimum value is 1 room. The current bill updates automatically as you change the room count.
4
Structure and Property Use Selection

How It Works:

  • Select a Structure from the dropdown (e.g., "Modern Building")
  • Property Use dropdown automatically populates with uses available for that structure
  • Select the Property Use (e.g., "Residential" or "Commercial")
  • System immediately calculates the fee per room based on structure + use combination
  • Adjust Number of Rooms using the +/- counter
  • Current Bill updates in real-time: Fee per Room × Number of Rooms
Real-Time Calculation: As you select structure, property use, and adjust room count, the system instantly displays the calculation breakdown (e.g., "GHS 75/room × 5 rooms for Modern Building - Residential") in a dynamic fee display panel.
5
Location Information Section

Location Capture (Same as Business):

  • Property Location - Enter address manually or capture via GPS
  • Click "Capture GPS Location" button
  • System uses high-accuracy GPS with fallback to standard accuracy
  • Coordinates auto-populate and can be manually adjusted
  • Location reverse-geocoded to show specific neighborhood/area

Zone Assignment:

  • Zone * - Select from dropdown (required)
  • Sub-Zone * - Select from dropdown (required, filtered by zone)
Note: Unlike business registration, BOTH zone and sub-zone are required for property registration.
6
Billing Information Section

Automatic Calculation Fields:

  • Current Bill - Automatically calculated as: Fee per Room × Number of Rooms (read-only field)
  • Amount Payable - Automatically calculated using formula: Old Bill + Arrears + Current Bill - Previous Payments (read-only field)

Manual Entry Fields (Optional):

  • Old Bill - Previous outstanding amount (defaults to 0.00)
  • Previous Payments - Amount already paid (defaults to 0.00)
  • Arrears - Outstanding arrears (defaults to 0.00)
  • Batch - Group identifier for batch processing
7
Form Validation and Submission

Before submission, the system validates:

  • Owner name is provided
  • Structure and property use are selected
  • Number of rooms is at least 1
  • Zone and sub-zone are both selected
  • GPS coordinates are valid (if provided)
  • Telephone format is correct (if provided)

Click "Register Property" to save. Upon successful registration, you'll be redirected to the property profile view page.

Example: 5-Room Residential Property Registration
1. Enter Owner Name: "John Mensah"
2. Enter Telephone: "+233 24 123 4567"
3. Select Gender: "Male"
4. Select Structure: "Modern Building"
5. Property Use auto-populates with "Residential" and "Commercial"
6. Select Property Use: "Residential"
7. Set Number of Rooms: Click + button to reach 5 rooms
8. System calculates: GHS 75.00 per room × 5 rooms = GHS 375.00
9. Dynamic display shows: "GHS 375.00 - GHS 75/room × 5 rooms for Modern Building - Residential"
10. Click GPS button, location captured: "Dzorwulu, Accra"
11. Select Zone: "Dzorwulu Zone"
12. Select Sub-Zone: "Dzorwulu East"
13. Amount Payable displays: GHS 375.00
14. Submit form

Property Number System

Format: Property numbers follow the format PROP000001, PROP000002, etc., and are automatically generated for unique identification.

Dynamic Billing Calculation

The system automatically calculates property bills based on:

  • Structure Type: Modern Building, Concrete Block, Mud Block, etc.
  • Property Use: Commercial properties typically have higher rates than residential
  • Room Count: Fee per room multiplied by total number of rooms
  • Real-Time Updates: Change any parameter (structure, use, or room count) and see instant calculation updates
Example Fee Structure:
Modern Building - Residential: GHS 75.00 per room
Modern Building - Commercial: GHS 150.00 per room
Concrete Block - Residential: GHS 50.00 per room
Concrete Block - Commercial: GHS 100.00 per room

5-Room Modern Commercial Property:
GHS 150.00 × 5 = GHS 750.00 current bill

Troubleshooting

Common Registration Issues

Business Type Not Selectable

  • Must select from dropdown, not just type
  • Wait for dropdown to appear before selecting
  • Clear field and start typing again
  • Contact admin if business type is missing

Category Dropdown Empty

  • Ensure business type was properly selected
  • Check that fee structure is configured for that type
  • Refresh page and try again
  • Contact administrator to verify fee structure

GPS Location Fails

  • Ensure location services are enabled
  • Allow browser location access
  • Move to area with better GPS signal
  • Manually enter coordinates if needed

Current Bill Shows Zero

  • Verify business type and category are selected
  • For properties, check structure and property use
  • Ensure room count is at least 1
  • Contact admin to check fee structure configuration

Common Login Issues

Cannot Login

  • Verify username and password accuracy
  • Check for caps lock activation
  • Contact administrator for password reset
  • Clear browser cache and cookies

First Login Password Change Fails

  • Ensure new password meets requirements
  • Confirm password confirmation matches
  • Try different browser
  • Contact system administrator

Bill Generation Problems

Solutions:

GPS Location Best Practices

For Accurate GPS Capture
Follow these tips for best results when capturing GPS locations.
  • Location Services: Ensure device location services are enabled before starting
  • Signal Quality: Capture outdoors or near windows for better GPS signal
  • Browser Permissions: Always allow location access when prompted
  • Wait for Accuracy: System tries high-accuracy GPS first, then falls back if needed
  • Verify Results: Check that geocoded address matches expected location
  • Manual Entry: Coordinates can be manually entered/adjusted if GPS fails
